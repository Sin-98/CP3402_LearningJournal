## Content Insights
This week I learned how child themes allow developers to customise an existing WordPress theme without modifying the parent theme directly. This is important because updates to the parent theme do not overwrite changes made in the child theme making it a safer and more maintainable approach.
Creating the child theme helped me understand the purpose of the style.css and functions.php files. The style.css file identifies the child theme and contains custom CSS, while the functions.php file loads the parent theme's stylesheet so that the child theme inherits the original design before applying additional customisations.
I also learned how browser developer tools can improve the development process. By using the browser inspector, I could experiment with CSS changes in real time before copying the final styles into my child theme. This reduced trial and error and made it much easier to create consistent customisations.
Completing the required visible modifications showed me how small CSS changes, such as adjusting colours, fonts, spacing, buttons, and layouts, can significantly improve the overall appearance and user experience of a website without changing its underlying functionality.

## Career/Employability/Learning Insights
Before this practical, I thought child themes were simply another way to organise CSS. After completing the exercise, I realised they represent a professional workflow that allows developers to customise client websites while keeping them maintainable and compatible with future updates. This is especially valuable for agencies that manage multiple WordPress websites over long periods.
While researching WordPress careers, I found that many junior web developer and WordPress developer roles specifically mention experience with custom themes, child themes, PHP, CSS, and troubleshooting existing websites. This showed me that understanding how to extend an existing system is just as important as building something from scratch. Even though I am more interested in software engineering than WordPress development, these skills strengthen my understanding of how large web applications are structured.
