## Content Insights
This week introduced me to the fundamentals of PHP and how server-side programming differs from HTML, CSS, and JavaScript. Unlike client-side code, PHP is executed on the server before the webpage is sent to the browser. This allows dynamic content to be generated based on data, user input, or application logic. I learned how to use echo to generate HTML, implement decision-making with if and else statements, repeat code using for, while, and foreach loops, create reusable functions with parameters, and organise code using include to share common headers and footers across multiple pages. Using include helped me understand how code reuse improves maintainability because shared components only need to be updated in one place.

## Career/Employability/Learning Insights
Learning PHP gave me a better appreciation of how dynamic websites work behind the scenes. Although modern web development often uses frameworks and other programming languages, PHP is still widely used because it powers WordPress and many existing business websites. Researching web development job advertisements showed that PHP continues to be listed alongside HTML, CSS, JavaScript, SQL, and Git for many junior web developer positions, particularly in companies that maintain or customise CMS-based websites.
One of the biggest lessons this week was recognising the value of writing reusable code. Initially, it seemed easier to copy and paste repeated HTML into multiple pages, but using include demonstrated why experienced developers avoid duplication. When common code is stored in a single file, future changes become much easier and there is less risk of introducing inconsistencies. This reinforced the software engineering principle of keeping code maintainable rather than simply making it work.
Going forward, I want to continue strengthening both my PHP skills and my understanding of software development workflows. While frontend development remains my strongest area, learning server-side programming has shown me how the frontend and backend work together to create complete web applications. Building knowledge across both areas will make me a more versatile developer and better prepared for future software engineering roles.
