## Content Insights
This week I learned the difference between creating a website from scratch and using a Content Management System (CMS) such as Joomla. Unlike manually coding every page in HTML and CSS, Joomla provides a dashboard that allows content, navigation menus, images, and layouts to be managed through a graphical interface.
I learned how Joomla organises website content through articles, categories, menus, and modules. Understanding the relationship between these components was important because simply creating a page does not automatically make it appear in the website navigation. Menu items must be created separately and linked to the correct content.
Another useful lesson was learning how templates control the overall appearance of a website. By modifying template settings, colours, typography, page layouts, and module positions, I was able to make my jewellery website look different from the default Joomla installation. This showed me how CMS platforms separate content from presentation, making websites easier to maintain and update.
I also learned that Joomla allows website functionality to be extended through extensions and plugins. While I only worked with basic features this week, I can see how additional functionality such as e-commerce, user accounts, and contact forms could be added later without needing to build everything from scratch.

## Career/Employability/Learning Insights
I viewed CMS platforms as tools mainly for people with limited programming knowledge. After using Joomla, I realised that professional web developers often use CMS platforms because they significantly reduce development time while still allowing customisation when needed.
I researched web development job advertisements and found that many positions require experience with CMS platforms such as WordPress, Joomla, or Drupal, especially for roles involving website maintenance and content management. This made me realise that CMS knowledge is a practical employability skill because many businesses prefer solutions that non-technical staff can update themselves.
When I customised the template, added pages, and experimented with menu settings while following tutorials, I understood the concepts much better. Going forward, I plan to continue using a learn-and-apply approach because it helps me retain information and solve problems more independently.
